🔧 Quick Setup

1

Extract & Install

tar -xzf eryzaa-*.tar.gz cd x86_64-unknown-linux-gnu/ sudo ./install.sh
2

Start Rental Server

./eryzaa-rental

Configure your hourly rate and share your ZeroTier IP

3

Connect as Client

./eryzaa-client

Access rental servers via SSH, AI training, or edge computing
